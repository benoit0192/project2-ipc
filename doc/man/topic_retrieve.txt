NAME
	topic_retrieve - retrieve a message from a topic.

SYNOPSIS
	#include ...

	int topic_retrieve(topicid_t id, char **buf)

DESCRIPTION
	topic_retrieve() allows a process to retrieve a message from a topic.

RETURN VALUE
	On success, buf is set to an allocated buffer containing a message, and the function returns the size of the message.

	If the specified topic doesn't exist, -1 is returned.
	If the process is not registered to retrieve messages from this topic, -2 is returned.
	If another error occurs, -3 is returned.
