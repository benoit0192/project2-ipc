NAME
	topic_unsubscribe - unregister a process to stop receiving messages published in a topic.

SYNOPSIS
	#include ...

	int topic_unsubscribe(topicid_t id)

DESCRIPTION
	topic_unsubscribe() allows a process to remove itself from the subscriber list of a topic. If the specified topic doesn't exist, it returns an error.

RETURN VALUE
	On success, zero is returned.

	If the specified topic doesn't exist, -1 is returned.
	If the user wasn't registered in this topic, -2 is returned.
	If another error occurs, -3 is returned.
